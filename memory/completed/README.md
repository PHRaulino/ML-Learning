# Tópicos concluídos

Arquivos de performance de tópicos **dominados** e já mergeados para `main`.

Registro permanente de:
- Quando foi estudado
- Quais assessments foram feitos e scores
- Pontos fortes consolidados
- Armadilhas identificadas naquele tópico

Serve de referência para:
- Revisões futuras (revisitar se aparecer 🔁 no roadmap)
- Contexto para tópicos dependentes (ex: ao estudar Random Forest, olhar `arvore-decisao.md`)
- Honestidade sobre o próprio progresso

## Não editar após merge

Arquivos aqui são imutáveis (exceto por revisões formais, ver abaixo).

## Revisão formal

Se uma revisão (🔁) detectar que o tópico precisa ser re-estudado, criar nova branch `topic/<nome>-revisao` e criar novo arquivo em `memory/performance/<nome>-revisao.md`. O arquivo original em `completed/` permanece como registro histórico.
