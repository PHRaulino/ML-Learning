# ML Learning

Laboratório pessoal de aprendizado de Machine Learning com metodologia **100% prática** e tutor-AI (Claude) que nunca entrega a resposta.

## Filosofia

Aprendo fazendo. Claude age como tutor socrático: pergunta, provoca, valida, mas **não resolve** os exercícios por mim. O objetivo é construir intuição real — não copiar código.

## Estrutura

- **`planning/`** — roadmap vivo, conceitos por dominar, assessments para medir nível.
- **`memory/`** — o que já estudei, como me saí, o que ainda trava.
- **`practice/`** — notebooks e labs usando dados reais do fintrack (banco pessoal).
- **`docs/`** — metodologia, workflow de branches, ferramentas disponíveis.

## Como usar

1. Comece lendo `docs/methodology.md` e `docs/workflow.md`.
2. Veja o foco atual em `planning/current-focus.md`.
3. Ao iniciar uma sessão com Claude, o contexto é carregado automaticamente via hook.
4. Cada novo tópico = nova branch. Só vai pra `main` quando dominado.

## Ferramentas

- Jupyter notebooks (`practice/notebooks/`)
- MCP postgres → banco do fintrack
- MCP fintrack → APIs de alto nível
- Git branches para isolar tópicos
